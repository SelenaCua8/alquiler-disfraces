# 🎭 Costume Rental System - University Project

This is a web application developed as part of my university coursework to learn **PHP** and **Relational Databases**. The system manages a costume rental business, covering everything from the frontend presentation to the backend logic and database persistence.

## 🚀 Project Overview

The main goal of this project was to implement a CRUD (Create, Read, Update, Delete) system using PHP and MySQL, focusing on clean code and database integrity.

## 🛠️ Tech Stack

* **Backend:** PHP 8.x
* **Database:** MySQL (managed via phpMyAdmin)
* **Frontend:** HTML5, CSS3
* **Environment:** Developed on **Ubuntu Linux** 🐧 using XAMPP/Apache.

## 📌 Key Features

* **Inventory Management:** Add, edit, and remove costumes from the catalog.
* **Rental Logic:** Manage rental dates and availability.
* **Database Integration:** Relational schema to store costumes, categories, and clients.
* **Responsive UI:** Basic styling for a clean user experience.

## 📂 Database Setup

1.  Open **phpMyAdmin**.
2.  Create a new database named `alquiler_disfraces` (or the one specified in your config).
3.  Import the `.sql` file located in the `db/` or root folder.

## ⚙️ Installation

1.  Clone the repository into your local server folder (e.g., `htdocs` or `/var/www/html`):
    ```bash
    git clone [https://github.com/SelenaCua8/alquiler-disfraces.git](https://github.com/SelenaCua8/alquiler-disfraces.git)
    ```
2.  Configure your database credentials in the connection file.
3.  Access the project via `localhost/alquiler-disfraces`.
